# CycleGraph Frontend

基于 Vue 3 + Vite + Element Plus 构建的前端界面。

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

## 技术栈

- Vue 3 - 渐进式JavaScript框架
- Vite - 下一代前端构建工具
- Element Plus - Vue 3 组件库
- Vue Router - 官方路由管理器
- Axios - HTTP 客户端

## 项目结构

```
frontend/
├── public/          # 静态资源
├── src/
│   ├── api/         # API 接口
│   ├── assets/      # 资源文件
│   ├── components/  # 组件
│   ├── router/      # 路由配置
│   ├── views/       # 页面视图
│   ├── App.vue      # 根组件
│   └── main.js      # 入口文件
├── index.html       # HTML 模板
├── package.json     # 项目配置
└── vite.config.js   # Vite 配置
```
